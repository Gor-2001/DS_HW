## Requirements

* Python 3.10 or higher
* NumPy
* Other libraries as needed

## Setup

1. **Create a virtual environment**

```bash
python3 -m venv .venv
```

2. **Activate the virtual environment**

```bash
source .venv/bin/activate
```

3. **Install dependencies**

```bash
pip install numpy
pip install <other-packages>
```

## Usage

You can run the project using the script:

```bash
source run.sh
```

Or run the problems individually:

```bash
python3 Problem_1.py
python3 Problem_2.py
...
python3 Problem_15.py
```

## Deactivate

To exit the virtual environment:

```bash
deactivate
```
